# Wproj

`wproj` is the project-level tool entry for Warp Parse. It is mainly used for workspace initialization, configuration checks, model inspection, runtime management, and selected data statistics.

Use `wproj` first when you need to:

- initialize or update a workspace
- check whether a project is complete and valid
- inspect models, collect statistics, or perform runtime management actions

## Recommended Reading

- Authoritative page: [`wproj` Project Tool Usage](../06-usage/cli/project.md)
- If you are just starting: read [Getting Started](../01-getting-started.md)
- If you want the overall CLI split: read [Warp Parse CLI Usage Guide](../06-usage/cli/index.md)

## Common Entry Points

```bash
wproj --help
wproj init --work-root .
wproj check --work-root .
```
